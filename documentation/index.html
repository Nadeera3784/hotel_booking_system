Introduction

Ultimate Hotel Booking System is an advanced booking system with real time availability and
room booking functionality.You can manage room, room types, capacity, hotel rates with different date range and month wise.

Features

  • Clean and Responsive Design
  • Assign unlimited number of room with combination of Room type and capacity.
  • Assign unlimited Room Type. i.e. Deluxe Room, Standard room.
  • Assign unlimited capacity. i.e. Single, Double, family.
  • Day Wise (Mon to Sun) price plan entry with regular and different date range.
  • Advance Payment (%) of total amount can be set month wise.
  • Active Booking list with cancellation feature and Booking history list.
  • Room Blocking in particular date range. i.e. booking from desk and phone blocking feature will be use.
  • 12 months calendar view of room availability.
  • Any currency support, different date format,  hotel  time zone setting,  minimum night booking setting,  price with tax and without tax setting.
  • After successful booking email  notification both hotel owner and customer  with invoice.
  • Confirmation Email notification
  • Room Type Image Upload
  • Booking list search by date range
  • Home  page widget for last 10 Booking, today check-in, today check-out.
  • multi currency supported
  • specail offer define by roomtype and minimum nights
  • child feature added.
  • Report, Booking and Financial(Weekly,Monthly,Yearly)
  • Nice calnedar view in fronend with availability and price.
  • Detailed Documentation

Requirements

  • Apache with PHP and MySQL server
  • PHP v5.6+
  • MySql
  • CURL enabled
  • date.timezone setting enabled

Installation

Step1:  Upload the downloaded zip file to your server in the public_html directory

Step 2:  Extract the zip file

Step3:  Create a new database from your server mysql.

Step4:  Open the file database.php from the directory /application/config/ database.php

Step5:  Fill up these informations with your database hostname, database username, database password, database name respectively which you have created in the previous step

Step6:  Now from server phpmyadmin go to your database. Select import and choose the file install.sql located in seeds/install.sql

Final Step:  You are ready to go now to browse the application

Your  admin panel:  http://www.yoursite.com/admin/

Default admin credentials are:

username : admin@admin.com
password : password

NOTE: After login to your dashboard change your password immediately

Payment Gateway setup :

Open the file database.php from the directory /application/libraries/ Paypal.php

get rid of commets on original URL and commets out sandbox URL

 

Dashboard

First Login in your Admin Panel. You can see following Option under "Dashboards"

  • Rooms Here you can insert number of room based on room type and capacity.
  • Room Type:  Here you can insert room type. i.e. Deluxe Room, Standard Room
  • Capacity: Here you can enter capacity title and number of occupency. i.e. Single, Double etc.
  • Gallery :  You can insert images based on roomtype and capacity

Price Manager

First Login in your Admin Panel. You can see following Option under "Price Manager"

  • Price Plan : Here you can edit default price plan and also you can add new price plan in specific date range. priceplan you need to enter day wise sun to sat.
  • Advance Payment:  Here you can set advance payement(%) of total amount need to pay customer at the time of booking. this feature can be enable and desable.
  • Specail offer: you can define specail offer by deduct price percent with minimum night and roomtype.

Booking

First Login in your Admin Panel. You can see following Option under "Booking"

  • Booking List:  Booking list devided into active booking & booking history. you need select type of booking need to list from dropdown combo. from active booking list you can print invoice, cancel booking, view booking details. From booking history you can booking booking details and delete booking forever.
  • Customer Lookup:  Here you can list of all customer details and can edit details. also you can view booking list of seelcted customer.      
  • Calendar:  Here you can see monthly calendar view of room availability and booked rooms.
  • Room Blocking:   Room Blocking in particular date range

How to set automatically remove expired bookings ?

Open the file App.php from the directory /application/controllers/App.php

Uncomment this line $this->common->clear_expired_bookings();

Currency

First Login in your Admin Panel. You can see following Option under "Currency"

you can add any currency which supported by openexchangerates.org. you must insert correct currency code, that is most important.

You can update exchange rate automatically by clicking update button on individual Currency

 

 

 

Reports

First Login in your Admin Panel. You can see following Option under "Reports"

  • Booking : Here you see your booking report weekly,monthly and yearly wise based on number of booking.
  • Financial : Here you see your financial report weekly,monthly and yearly wise based on number of booking.

Settings

First Login in your Admin Panel. You can see following Option under "Settings"

  • Global : Here you can manage Booking engine off and on, hotel time zone. minimum booking night set, date format, room locking time, tax setup.
  • Payment Geteway : Here you can update paypal email
  • Change Password : Here you can change your admin password

Language

How to add new language?
  • Please go to \application\language directory 
  • Copy the english language folder and paste (set the directory name for your language)
  • Add your translations in app_lang.php/ion_auth_lang.php/auth_lang.php
  • Finally set this language in \application\config\config.php
    $config['language'] = to your lang file

How Front-end work?

Step 1 : Select check-in date and check-out date and  Adult / Room.

step 2 : Search result will display with price. just select number of room you want from dropdown combo. then click continue.

Step 3 : Just enter customer details and select payemnt gateway.

step 4 :  Booking complete.
